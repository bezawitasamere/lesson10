About me

My name is Bezawit Asamere. I have taken an interest in talking about social media and how it affects us. During the pandemic I saw just how harmful it was to be in one's own bubble and not hear different opinions. We all had free time during the pandemic and were spending an inordinate amount of time online. It all starts with a harmless comment then it keeps getting worse and worse. The US vs THEM mentality was at its peak and I think social media is mostly to blame for this.

Social Media

I have been using social media for several years now. I have tried each and every form of social media without bias. I would spend a lot of time on my phone so I would like to be entertained, informed,..etc This is why I use different platforms but I have stopped using some. I would also like to add that social media has created some biases that I did not I had. It makes me feel like I am in an echo chamber since it only brings content I interact with. This is true for all social media except for pinterest.
